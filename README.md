## 화면구현

첫번째화면
- TableView 구현
- TableViewCell 구현
- HeaderView 구현
- Model 구현
- UI 구현 (image, label, view)
- AutoLayout을 사용한 동적크기 조정
- 저장된 영상 삭제

두번째화면
- 카메라영상 녹화 / 저장
- 원하는 방향으로 영상촬영 가능
- 카메라영상 녹화후 저장된파일을 filemanager로 통하여 로컬 저장공간에 영상을 저장

세번째화면
- 녹화된 영상
- 영상 컨트롤러 구현
녹화 영상의 토글(재생↔일시정지) 버튼
녹화된 영상의 전체 시간
현재 진행된 시간을 표시하는 진행바
